# FUTURE_UIUX_03
For this project, I designed a CRM dashboard for a small digital marketing agency that manages multiple clients and leads on a daily basis. The main goal was to simplify how the team tracks leads, manages client information, and handles follow-ups in one place. 

While designing, I focused on keeping the layout clean and easy to navigate. I used a sidebar for main sections like Dashboard, Leads, Clients, and Tasks, so users can quickly switch between different areas without confusion. 

One of the key features is the lead pipeline, which is divided into stages like New, Contacted, and Converted. This helps the team clearly see where each lead stands and makes it easier to manage progress visually instead of relying on scattered notes or messages. 

I also included summary cards at the top to give a quick overview of important numbers like total leads, active clients, and pending tasks. This allows users to understand the overall situation at a glance without going into detail. 

The client list section helps organize all clients along with their status and last interaction, making follow-ups more structured. Additionally, I designed a separate client profile screen where all important details like project information, notes, and activity history can be viewed in one place. 

Overall, my focus was on creating a simple, structured, and practical design that improves clarity and reduces confusion for teams managing multiple clients. The design aims to make daily workflows smoother and more efficient.
